An Avatar SDK

Botlibre as an SDK for avatars. This is the existing open source alternative to our Wave2Lip. This is also a viable alternative to rasa and it could also be used alongside rasa.

A Platform For The Users’ To Train Their Bots

Botfront this code has an UI interface and a training-deploying pipeline for the users’ to train their bots. This is an meteor application packaged as a docker container. More, relevant information and tools (like chat UI) regarding botfront can be found here. Botfront is also open source and its code is publicly available.

Alternatives to botfront: Botmock.
